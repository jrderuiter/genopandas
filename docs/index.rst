==========
GenoPandas
==========

GenoPandas is a library that extends basic pandas DataFrames to provide
additional functionality for handling various genomic data formats.

**Disclaimer: both library and documentation are works-in-progress and are
under active development. The API and other details may therefore be subject
to change.**
